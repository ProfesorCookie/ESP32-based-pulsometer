# ESP-32-based heart rate monitor  

Compact heart rate monitor based on mini ESP32 displaying BMP, Sp02 and graph.
# Components
- mini ESP32
- Adafruit SSD1306 0,96" OLED display
- MAX30102 sensor
- 10uF capacitor
- DIY box
